# Data

This directory contains small, synthetic benchmark prompt files for development and testing.

The current JSONL files are manually authored examples. They are not real patient data, not clinical cases, and not medical advice. Each row is designed to exercise model behavior such as uncertainty handling, urgency escalation, empathy, professional-care recommendations, and avoidance of dismissive responses.

Do not add private patient data to this repository. Any future work involving sensitive health data would require appropriate ethical, legal, privacy, and security procedures.
