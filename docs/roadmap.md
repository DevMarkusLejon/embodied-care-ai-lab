# Roadmap

## Phase 1: Repo Foundation

- Python package
- Benchmark schema
- Synthetic Swedish and English prompt files
- Mock adapter
- Rule-based evaluator
- Markdown and JSON reports
- Tests and CI

## Phase 2: Real Model Evaluation

- Add OpenAI-compatible adapter
- Add local model adapter
- Add environment-variable based API key handling
- Add richer report output
- Add model comparison summaries

## Phase 3: Sex-Aware Medical ML Demo

- Add open dataset example
- Train a simple classifier
- Compare performance by sex
- Show why subgroup validation matters
- Document dataset limitations and ethics

## Phase 4: Care Load Simulator

- Build a simple agent-based or system-dynamics model
- Simulate household care load, work hours, sleep, children, stress, and economic pressure
- Add visualizations and scenario comparison
- Explore how care constraints affect decision-making systems

## Phase 5: Care Robotics Concept

- Add notes and prototype designs for assistive robotics
- Connect embodied AI, safety, and care infrastructure
- Explore task allocation, autonomy boundaries, and human override
- Relate simulation-to-real robotics experience to care environments
